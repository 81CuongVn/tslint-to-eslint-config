# tslint-to-eslint-config

Converts your TSLint configuration to the closest possible ESLint equivalent.

## Usage

```shell
npx tslint-to-eslint-config
```

_(wow, so simple!)_

### `config`

```shell
npx tslint-to-eslint-config  --config ./path/to/tslint.json
```
